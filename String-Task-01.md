# Python Practice Task: Student Information Formatter

### Problem Statement

Create a Python program that:

1. Takes the following inputs from the user:

   * Student Name
   * Course Name
   * Age

 Display:

   * Student Name in **UPPERCASE**
   * Student Name in **lowercase**
   * Student Name in **Title Case**
   * Student Name with **Capitalize**

4. Print the data type of each variable.

### Sample Input

```python
Enter Student Name: rahul kumar
Enter Course Name: python programming
Enter Age: 20
```

### Expected Output

```python
Original Name: rahul kumar

Upper Case: RAHUL KUMAR
Lower Case: rahul kumar
Title Case: Rahul Kumar
Capitalize: Rahul kumar

Data Types:
Student Name: <class 'str'>
Course Name: <class 'str'>
Age: <class 'int'>
```
