* `name`


### Task 5: Rectangle Class

Create a class `Rectangle` with:

* `length`
* `width`

Create two methods:

```python
area()
perimeter()
```

Calculate and display the area and perimeter.

**Formula:**

```text
Area = length × width

Perimeter = 2 × (length + width)
```
